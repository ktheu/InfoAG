B01: Implementiere eine Klasse Kachel

  • Attribute: x, y für Position, size für Seitenlänge
  • Methode: display
In der display-Methode sorgt die Kachel dafür, dass sie mit schwarzer Umrandung, Randbreite 1 und keiner Füllung erscheint. Überprüfe die Werte der Attribute der Objekte k1 und k2 in der Konsole. 

    let k1, k2;

    function setup() {
      createCanvas(300, 300);
      k1 = new Kachel(30,200,40);
      k2 = new Kachel(50,80,100);
    }

    function draw() {
      background(230);
      k1.display();
      k2.display();
