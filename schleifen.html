Zwei geschachtelte for-Schleifen. Code

Linienmuster

Von einem zweidimensionalen Gitter führen alle Linien zu einem Punkt, dessen y-Koordinate von der y-Position der Maus abhängt. Die Gitterbreite hängt von der x-Position des Maus ab. Code

Kreismuster

Auf ein zweidimensionales Gitter werden Kreise gezeichnete. Die Gitterbreite hängt von der x-Position der Maus ab, der Kreisdurchmesser von der y-Position der Maus. Code

Dickemuster

Auf ein zweidimensionales Gitter werden Kreise gezeichnete. Die Gitterbreite hängt von der x-Position der Maus ab. Richtung unten rechts erhöht sich die Liniendicke. Code

Strichmuster

Auf ein zweidimensionales Gitter werden kleine Striche gezeichnete. Die Gitterbreite hängt von der x-Position der Maus ab. Die Anzahl der Striche von der y-Position der Maus. Ein Fall für drei geschachtelte for-Schleifen. Code
