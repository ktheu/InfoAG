Hinweise

Source Code in Chrome anschauen

Um den Source Code eines Sketches anzuschauen, gehe wie folgt vor:

  1. Gehe in Developer-Mode mit F12 oder Strg+Umschalt+i.
  2. Wähle Elements (1), dann klicke auf den Pfeil (2) in der linken oberen Ecke des Developer-Fensters und selektiere mit der Maus den Canvas des Sketches, der dich interessiert.
  3. In dem Developer-Fenster erscheint ein Abschnitt mit einer html-Datei (4), die meist so ähnlich heißt wie der Titel des Sketches.
  4. Wähle Sources (5). Mit Strg-P (6) wird eine Liste von Dateien angezeigt. Wähle die aus, deren Source-Code dich interessiert. Punkte 2 und 3 kann man weglassen, wenn aus dem Dateinamen zu erkennen ist, welcher Sketch es ist.

Sketche entwickeln mit Notepad++

Die Vorlage in eine *.html - Datei kopieren, mit Notepad++ den Sketch editieren und dann mit Ausführen, Launch in Chrome das Ergebnis ansehen. Dies geht für Sketche ohne Bilder oder Sound. Nach jedem Edit in der html-Datei den Browser refreshen. Die Konsole erhält man in Chrome mit F12 oder Strg+Umschalt+i, Console.
